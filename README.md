<h1 align="center">go skeleton</h1>

# Introductions

Based on the original skeleton program of go, the purpose is to quickly start go and standardize project management.

# Installation

```
$ git clone https://github.com/zhanbai/gos.git project-name
```

# Quick start

```
$ cd project-name
# run main.go and visit 0.0.0.0:8080/hello (for windows "localhost:8080/hello") on browser
$ go run main.go
```
